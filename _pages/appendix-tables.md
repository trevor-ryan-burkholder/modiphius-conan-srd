## Zones and movement

| Zone        | Description                       |
| ----------- | --------------------------------- |
| Reach       | Within arm's length; melee.       |
| Close       | A few steps.                      |
| Medium      | Across a room.                    |
| Long        | Distant but visible.              |
| Extreme     | At the edge of useful action.     |

| Movement              | Effect                                        |
| --------------------- | --------------------------------------------- |
| Move (1 free / turn) | Shift 1 zone, or reposition within zone.       |
| Sprint                | Move 2 zones; cost 1 Momentum.                |
| Run                   | Move 3 zones; no Standard Action that turn.   |

## Weapon Qualities — short forms

| Quality            | Effect (short)                                              |
| ------------------ | ----------------------------------------------------------- |
| Parrying           | Counts as shield for Parry.                                 |
| Vicious X          | +X damage per Effect rolled.                                |
| Piercing X         | Reduces target Soak by X.                                   |
| Knockdown          | First Effect rolled → target prone.                         |
| Stun               | First Effect rolled → target +1 difficulty on next test.    |
| Intense            | Damage applies to Resolve instead of Vigor.                 |
| Fearsome X         | On Wound, also X [CD] Resolve damage.                       |
| Reach              | Close-range targets count as Reach.                         |
| Unbalanced         | No Parry Reaction after use until next turn.                |
| Hidden X           | Observation to spot is +X difficulty.                       |
| Volley             | Area attack across a formation.                             |
| Thrown             | Throwable; uses Ranged Weapons.                             |
| Reload Minor/Std   | Requires Minor / Standard Action to reload.                 |
| Improvised         | +1 difficulty to attack; fragile.                           |

## Damage thresholds and wounds

| State                                  | Effect                                              |
| -------------------------------------- | --------------------------------------------------- |
| Vigor or Resolve hits 0                | Next damage → Wound or Trauma; track resets.        |
| 1 Wound (or 1 Trauma)                  | +1 difficulty to physical (or mental) tests.        |
| 2 Wounds / Traumas                     | As above; each test costs 1 Momentum/Doom to attempt. |
| 3+ Wounds                              | **Dying.** Stabilize in 3 rounds or die.            |
| 3+ Traumas                             | **Broken.** Stabilize, or rewrite the character.    |
